# Drop Bot (Farcaster + Telegram)

Этот бот автоматически отслеживает посты в Farcaster (через Neynar API) по ключевым словам (например, "drop address", "wallet") и автоматически отвечает вашим Ethereum-адресом. Также отправляет уведомления в Telegram.

## Настройка

1. Склонируйте репозиторий.
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Скопируйте `.env` и заполните его:
   - `NEYNAR_API_KEY`: Получите на [dev.neynar.com](https://dev.neynar.com/).
   - `NEYNAR_SIGNER_UUID`: UUID одобренного подписанта (Signer) для вашего аккаунта Farcaster.
   - `ETH_ADDRESS`: Ваш кошелек для получения дропов (рекомендуется Base L2).
   - `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_ID`: Для уведомлений.

## Запуск

```bash
npm start
```

## Как получить Signer UUID?

Для отправки ответов в Farcaster вам нужен "Signer". Вы можете создать его через панель управления Neynar или используя их SDK. Помните, что создание Signer требует подтверждения через мобильное приложение Warpcast и небольшого количества ETH в сети Optimism (обычно около $2) для регистрации ключа в протоколе.

## Дисклеймер

Используйте на свой страх и риск. Слишком частые ответы могут привести к бану аккаунта в Farcaster.
